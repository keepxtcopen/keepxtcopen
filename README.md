# keepxtcopen

我们只讲事实、不说白话，只为让大家知道小天才公司的所做所为。

**小天才做了什么——**

| 问题 | 说明 |
|------|------|
| 违反 GPL 协议 | 手表使用 Linux 内核，但从未公开内核源码 |
| 霸王条款 | 用户协议允许单方面修改、服务随时终止且不退款 |
| 云控封号 | root 或安装第三方软件直接封号，协议未明确禁止 |
| 社交垄断 | 微聊系统封闭，利用市场份额绑架儿童社交 |
| 1.20 事件 | 未经同意砍好友圈功能，删好友连带删点赞 |
| 维权指南 | 普通用户、开发者、被封号用户分别能做什么 |

这不是什么商业竞争抹黑。这里面每一条都有事实依据，每一条都是普通用户自己发现的。

## 关于这个项目

本项目不玩情怀、不搞虚头巴脑的口号，只基于真实设备行为、底层逻辑与公开现实，如实呈现小天才系列设备的封闭机制、权限管控、账号封禁策略、内核闭源现状与生态壁垒问题。

不刻意引战，也不刻意洗白。只把普通人看不到、被刻意隐藏的东西公开透明化。

让每一位使用者，都能清楚知道：自己花钱购买的设备，到底被限制了什么、被管控了什么、被剥夺了什么权利。
