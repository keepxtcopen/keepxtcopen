## Sekilas Info

Kalau kamu baru pertama kali dengar hal-hal ini, ini versi singkatnya:

**Apa yang dilakukan Xiaotiancai:**

1. Sistem arloji berbasis Android (kernel Linux), tapi tidak pernah merilis kode sumber kernel sesuai ketentuan GPL -- ini bermasalah secara hukum
2. Perjanjian pengguna berisi banyak klausul yang menguntungkan perusahaan, termasuk hak untuk mengubah perjanjian sepihak, membatasi jalur pengguna untuk menegakkan hak, dll.
3. Perangkat memiliki sistem kontrol cloud, yang mendeteksi root, pemasangan aplikasi pihak ketiga, dll. dan langsung memblokir akun, tanpa pemberitahuan yang jelas dalam perjanjian
4. Sistem Chat mikro (Weiliao) hanya mengizinkan pertemanan antar sesama perangkat Xiaotiancai, memanfaatkan posisi pasar dominan untuk menciptakan hambatan sosial, pada dasarnya menyandera hubungan sosial anak-anak
