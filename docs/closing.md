# 最后说几句

> 写这个项目没什么宏大目标。只是把一些散落在中文互联网各个角落的信息拼到了一起。

---

我本人不是任何竞品的人，只是关注消费电子和开源领域的时间比较长，陆陆续续看到了很多关于小天才的讨论。

这些讨论分布在知乎、酷安、贴吧、B站评论区、什么值得买……它们没有一个集中的地方。我就想着干脆整理一下。

## 为什么叫 keepxtcopen

xtc = 小天才拼音缩写。keep ... open = 保持公开透明。

没什么深意。就是觉得这些事情应该被摆到台面上来说。

有人可能要问

> 小天才**根本没有**开放过，为什么要keep呢

我对此的解释是：曾经小天才有过相对开放的时候（v1时代），虽然并不是绝对上的开放

当然你如果硬要辩解，我只能说，这个项目名字不重要

## 关于公平

这个项目里写的东西，我不保证100%准确。毕竟我不是小天才内部的人，很多信息是从用户反馈、行业常识、逻辑推理来的。

如果你是小天才的人，觉得我写得不对，欢迎指正。

但请记住：**我没有义务帮你证明小天才没有问题。你有义务证明小天才没有问题。**

如果你真能做到公开内核源码、修改用户协议、取消不合理的封号机制、开放社交协议——那我第一个把这个项目删了，然后给你道歉。

## 关于"为什么不直接维权"

经常有人说："觉得不好就去起诉啊""有本事别买啊""又没人逼你买"

说实话，这些说法都忽略了一个基本事实：

**普通消费者面对一个市场份额巨大的公司，维权成本极高。**

- 走法律途径：时间、金钱、精力
- 不买：孩子被社交圈排斥（这不是孩子能选择的）
- 忍着：总觉得哪里不对，但又说不出具体哪里不对

这个项目的目的不是什么"打倒小天才"。而是把那些"说不出来哪里不对"的东西，写成文字，让每个人都能看到、能理解。
