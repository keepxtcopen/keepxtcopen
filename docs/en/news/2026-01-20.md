# January 20 Incident: Friends Circle Overhaul

On January 20, 2026, Xiaotiancai made a series of changes to the Friends Circle feature without advance notice, sparking widespread user dissatisfaction.

**Major changes:**

- Friends Circle likes now visible only to mutual friends
- All historical likes disappear after unfriending
- Benefit cards renamed "Genius PIN," gifting removed
- **Later addition:** Level system permanently removed
- **Later addition:** Main page likes replaced with "Sports Encouragement"

None of these changes offer users any choice.
