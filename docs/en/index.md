## Quick Overview

If this is your first time hearing about these things, here's the short version:

**What Xiaotiancai did:**

1. The watch system is based on Android (Linux kernel), but has never released kernel source code as required by the GPL — this is legally problematic
2. The user agreement contains numerous company-favoring clauses, including the right to unilaterally modify the agreement and restricting users' avenues for legal recourse
3. Devices have a cloud-based control system that directly bans accounts when it detects root access or third-party app installation, without clearly disclosing this in the agreement
4. The micro-chat system only allows adding friends between Xiaotiancai devices, using market dominance to create social barriers — essentially holding children's social relationships hostage
