# 我該怎麼做

> 光罵沒用，得知道能做什麼。下面按不同身份整理了能做的事。

## 作為普通消費者

### 1. 買之前想清楚

如果你正在考慮給孩子買小天才手錶，問自己幾個問題：

- **你買手錶是為了什麼？** 如果只是為了定位和通話，那市面上能做到這個功能的手錶很多，不一定非選小天才
- **孩子班上是不是都在用小天才？** 如果是，你要做好準備——買了其他品牌，孩子可能沒法透過手錶加同學好友
- **你能接受裝置不受自己控制嗎？** 小天才的裝置有遠端管控能力，理論上廠商可以隨時決定你的裝置能做什麼、不能做什麼
- **你能接受哪天功能被砍了、帳號被封了都沒地方說理嗎？** 使用者協議偏向公司，維權渠道有限

這些問題沒有標準答案。每個人的情況不一樣。但至少應該在買之前知道這些，而不是買了之後才發現。

### 2. 如果已經買了

- **不要把家長App的權限完全交給孩子。** 小天才的裝置管控最終是透過家長端實現的，家長帳號是最高權限
- **定期檢查使用者協議和隱私政策有沒有更新。** 雖然沒人會天天看，但至少大版本更新時掃一眼
- **不要嘗試 root 或安裝非官方應用。** 在當前的政策下，這會導致封號且幾乎沒有申訴渠道
- **有問題先保留證據。** 截圖、錄屏、客服對話記錄，萬一走到維權那一步用得上

### 3. 考慮替代方案

如果你確定不想買小天才，市面上還有這些選擇（不構成推薦，只是列舉）：

- **華為兒童手錶** — 生態相對開放，但社交也是僅限於華為裝置
- **小米米兔手錶** — 價效比路線，功能相對基礎
- **360 兒童手錶** — 比較老牌的玩家，市場份額不如小天才
- **步步高旗下其他品牌** — 嗯，跟小天才其實是一家的

> 說實話，目前國內兒童手錶市場在社交互通上基本是各玩各的。這不是小天才一家的問題，是整個行業的問題。只不過小天才份額最大、做得最絕。

## 作為開發者 / 開源愛好者

### 1. 向 GNU FSF 舉報 GPL 違規

小天才手錶使用了 Linux 內核但沒有公開原始碼。你可以向 **Free Software Foundation** 或 **Software Freedom Conservancy**（SFC）舉報。

GNU 官網有專門的 GPL 合規頁面：

- **GNU GPL Compliance** — https://www.gnu.org/licenses/gpl-compliance.html
- **Software Freedom Conservancy** — https://sfconservancy.org/

舉報時需要提供：

- 裝置型號和購買的 Android 版本
- 證明裝置使用了 Linux 內核的證據（比如內核版本號截圖）
- 證明廠商沒有提供原始碼的證據（比如在官網和 GitHub 上搜尋無結果）

SFC 過去成功推動過很多公司（如 LG、三星、思科）遵守 GPL。雖然他們主要關注海外市場，但中英文舉報他們都受理。

如果你也想加入我們，歡迎fork本專案、提Pull Request

### 2. 在 GitHub 上關注和推進

- 搜尋「xiaotiancai kernel」或「imoo kernel source」相關 issue
- 如果有開發者已經在做反編譯或內核分析的工作，關注並參與
- 如果小天才某天發布了內核原始碼，檢查其合規性

### 3. 幫助普通使用者理解技術問題

- 在論壇裡幫人解釋什麼是 GPL、為什麼閉源有問題
- 翻譯和傳播相關技術內容
- 如果有能力，可以嘗試分析裝置韌體並公開發現

## 作為被雲控封號的使用者

### 1. 先確認自己到底違反了哪條規則

說實話，很多時候使用者自己都不知道為什麼被封。常見的情況：

- 透過 adb 安裝過 APK
- 嘗試過取得 root 權限
- 繫結過bot
- 存在逆向並且呼叫API的行為 <-（作者經歷）
- 被系統誤判（較少見）

**但是關鍵問題是：這些「規則」在使用者協議裡並沒有明確寫出來。**

所以你首先是「被一個你不知道的規則處罰了」。

### 2. 維權渠道

**第一步：找客服**

雖然大概率沒結果，但這是走其他渠道的前提——你得先證明你找過廠商協商了。

- 小天才官方客服熱線：400-610-3999
- 保留所有通話錄音和聊天記錄

**第二步：12315 投訴**

全國統一的消費者投訴熱線。可以在微信小程式搜「全國12315平臺」直接投訴。

投訴要點：

- 購買的裝置因廠商單方面判定違規而被限制正常使用
- 使用者協議中未明確列出封號的具體規則
- 封號導致裝置無法正常使用，影響了孩子的正常通訊和安全

**第三步：工信部投訴**

如果涉及通訊功能被限制，或者上一步維權失敗，可以向工信部電信使用者申訴受理中心投訴。

網址：https://www.chinatcc.gov.cn/

**第四步：媒體曝光**

- 在黑貓投訴（https://tousu.sina.com.cn/）上發帖
- 在知乎發帖講自己的經歷
- 聯絡數碼類博主或自媒體

小天才對品牌形象還是很在意的。單個使用者維權可能沒人在乎，但如果話題發酵了，他們會有壓力。

### 3. 集體維權的可能性

兒童手錶封號這個問題不是個例。如果你被封了，大概率還有很多人跟你一樣。

可以嘗試：

- 在酷安、貼吧、知乎上找同樣遭遇的人
- 集體向 12315 或工信部投訴（集體投訴比單個投訴更有效）
- 如果有律師資源，可以諮詢是否構成集體訴訟的條件
