## 快速了解

如果你是第一次聽說這些事情，這裡給你一個極簡版：

**小天才做了什麼：**

1. 手錶系統基於安卓（Linux內核），但從未按照GPL協議要求公開內核原始碼 — 這在法律上是有問題的
2. 使用者協議裡有大量偏向公司的條款，包括單方面修改協議的權利、限制使用者維權途徑等
3. 裝置有雲端管控系統，偵測到root、安裝第三方App等情況會直接封禁帳號，且協議中並未明確告知
4. 微聊系統只允許小天才裝置之間加好友，利用市場優勢地位製造社交壁壘，本質是綁架兒童社交關係
