## Mabilis na Pag-intindi

Kung ito ang unang pagkakataon na marinig mo ang mga bagay na ito, narito ang pinakasimpleng bersyon:

**Ano ang ginawa ng Xiaotiancai:**

1. Ang sistema ng relo ay batay sa Android (Linux kernel), ngunit hindi kailanman inilabas ang kernel source code ayon sa kinakailangan ng GPL — ito ay may problema sa batas
2. Ang kasunduan ng user ay naglalaman ng maraming probisyon na pabor sa kumpanya, kabilang ang karapatang baguhin ang kasunduan nang unilateral, paghihigpit sa paraan ng pagtatanggol ng mga user, atbp.
3. Ang device ay may cloud-based control system na direktang magba-ban sa account kapag may nakitang root o pag-install ng third-party na app, at hindi ito malinaw na isinasaad sa kasunduan
4. Ang WeChat-like chat system ay nagpapahintulot lamang ng pag-add ng kaibigan sa pagitan ng Xiaotiancai device, na gumagamit ng dominanteng posisyon sa merkado upang lumikha ng social barrier — ito ay pag-hostage sa ugnayang panlipunan ng mga bata
