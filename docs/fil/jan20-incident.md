# Pangyayari noong 1.20: Malaking Pagbabago sa Friend Circle

> Noong Enero 20, 2026, nang walang anumang abiso, ang Xiaotiancai ay nagsagawa ng serye ng mga pagbabago sa Friend Circle function. Walang anumang paghahanda ang mga user, nagising sila at nakita na lahat ay nagbago.

Ito ay hindi pag-aayos ng bug, ito ay sapilitang pagtanggal ng mga umiiral na function ng user. At ginawa ito nang hindi alam o pumayag ang user.

---

## Pagbabago Uno: Pagbawas ng Visibility ng Likes sa Friend Circle

**Bago ang pagbabago:** Ang mga like ng sinuman sa Friend Circle ay makikita mo. Kahit hindi mo kaibigan ang nag-post o hindi mo kilala ang nag-like, ang like na iyon ay naka-display doon.

**Pagkatapos ng pagbabago:** Tanging ang mga like ng **mutual friends** ang makikita. Ang mga like ng hindi mutual friends ay direktang nakatago.

Sa ibabaw, ito ay para sa "privacy protection," ngunit sa katunayan, isa itong sapilitang pagkastrat ng isang function na dating bukas at may social na katangian.

Ano ang dahilan ng pagkakaroon ng Friend Circle? Para makita ng magkakaibigan ang interaksyon ng bawat isa. Kung itatago mo ang mga like ng hindi mutual friends, mawawala ang malaking bahagi ng dahilan ng pagkakaroon ng Friend Circle.

**At walang pagpipilian.** Walang switch, walang setting, walang opsyon na "gusto mo bang makita ng hindi kaibigan ang iyong like." One-size-fits-all, direktang binago.

---

## Pagbabago Dalawa: Mawawala ang Like Pagkatapos Magtanggal ng Kaibigan

**Bago ang pagbabago:** Nag-post ka, may nag-like. Kahit tanggalin mo ang taong iyon mamaya, nandoon pa rin ang kanyang like.

**Pagkatapos ng pagbabago:** Tanggalin ang kaibigan, lahat ng like na ibinigay ng taong iyon sa iyo ay kakanselahin.

Ang pagbabagong ito ay pinakamalaki ang epekto sa mga batang mahilig mag-ipon ng likes. Maaaring inabot ka ng ilang buwan upang makaipon ng daan-daang likes, tapos dahil nakipag-away ka sa isang tao at tinanggal mo siya — *puff*, lahat nawala.

Ito ay hindi teknikal na limitasyon, ito ay sadyang idinisenyo nang ganito.

Pag-isipan mo: **Tanggalin ang kaibigan = tanggalin ang kasaysayan ng interaksyon.** Ilagay mo ang lohikang ito sa WeChat, QQ, Instagram, walang isang social product ang gagawa nito. Dahil ang like aynakalakip data ng content mismo, hindinakalakip data ng "relasyon ng magkaibigan."

Bakit ganito ang pagbabago ng Xiaotiancai? Ang tanging makatwirang paliwanag ay: **Pukawin ang user na huwag basta-basta magtanggal ng kaibigan, panatilihin ang "stickiness" ng relasyon.** O kaya ay hindi lang nila pinag-isipan, naisip lang nila na "kung tinanggal, hindi na dapat ipakita."

Alinmang dahilan, ang user ang nagdurusa.

---

## Pagbabago Tatlo: Palitan ang Pangalan ng "QuanYiKa" sa "Tiancai PIN," Tanggalin ang Feature ng Pagbibigay

**Bago ang pagbabago:** Ang user ay makakakuha ng mga QuanYiKa sa pamamagitan ng iba't ibang aktibidad, at maaari ring ibigay ang QuanYiKa sa kaibigan.

**Pagkatapos ng pagbabago:** QuanYiKa ay pinalitan ng pangalan na "Tiancai PIN," hindi na sinusuportahan ang pagbibigay.

Ang QuanYiKa ay isang bagay na pinaghirapan ng user — oras o pagsisikap ang kapalit. Ang pagbibigay nito sa kaibigan ay sa isang banda ay karapatan ng user na itapon ang kanilang "asset." Sa ngayon, direktang tinatanggal ang function ng pagbibigay, na parang sinasabi:

**"Ang mga bagay na pinaghirapan mo, kami ang magpapasya kung paano gagamitin."**

Ang pagpapalit ng pangalan mismo ay walang problema — hindi naman maganda ang "Tiancai PIN" — ngunit ang paggamit ng pagpapalit ng pangalan upang tanggalin ang function ng pagbibigay, iyon ay medyo nakakadiri.

---

## Pagbabago Apat: Permanenteng Pagtatanggal ng Level System

Ang Xiaotiancai watch ay may **level system**. Ang user ay nag-level up sa pamamagitan ng araw-araw na paggamit, interaksyon, at pagkumpleto ng mga gawain. Ang level ay hindi lamang isang identity marker, may kasama rin itong mga karapatan at sense of achievement.

Para sa mga user na lumaki gamit ang Xiaotiancai watch, ang level ay isang patunay ng kanilang paglaki. Mula sa LV.1 hanggang sa mataas na antas, ang prosesong ito ay maaaring tumagal ng ilang taon.

At bigla na lang tinanggal ng Xiaotiancai.

**Permanenteng pagtatanggal, hindi suspensyon, hindi pagbabago ng bersyon, direktang pagputol.**

Ang level na inipon mo sa loob ng ilang taon, sa isang gabi ay nawala lahat. Walang anumang kompensasyon, walang anumang kapalit.

"Ang pagkabata ng isang henerasyon ay nagtatapos dito" — hindi ito eksaherado.

Para sa mga batang ipinanganak noong 2010s at 2015s, ang level, Friend Circle, at likes sa Xiaotiancai watch ay ang kanilang "QQ Space." Subukan mong balikan at i-zero ang lahat ng QQ level ng mga 80s at 90s? Tingnan natin kung sasabog.

Ngunit dahil ito ay para sa mga bata, iniisip ng matatanda na "level lang naman iyon," kaya tinanggal na lang nila, walang nagsasalita para sa kanila.

---

## Pagbabago Lima: Palitan ang Main Page Likes ng "Sports Encouragement"

**Dating function:** May like function sa main page ng Friend Circle, katulad ngFriend Circle like. Nag-post ka ng isang update, maaaring mag-like ang kaibigan mo, ito ang pinaka-basic na social interaction.

**Pagkatapos ng pagbabago:** Ang main page likes ay tinanggal, pinalitan ng isang function na tinatawag na "Sports Encouragement."

Ang tinatawag na sports encouragement, sa madaling salita, ay pwedeng mag-like lamang ng sports-related posts. Gusto mong mag-like sa araw-araw na post ng kaibigan mo? Hindi pwede. Ngayon, "encourage" na lang ng sports ang pinapayagan.

Ang lohika ng pagbabagong ito ay napakalinaw: **Gusto ng Xiaotiancai na gawing "sports check-in platform" ang Friend Circle mula sa "social platform."**

Iniisip nila na ang mga bata ay dapat gumamit ng relo para mag-ehersisyo at maging malusog, hindi para makipag-socialize. Kaya tinanggal nila ang social function at pinalitan ng "sports encouragement."

Ngunit ang problema ay — **ito ay hindi gusto ng magulang, hindi rin gusto ng bata, ito ay naisip lang ng product manager.**

Ang bata ay gumagamit ng relo upang makipag-ugnayan sa kaklase, iyon ay isang social need. Pipilitin mong gawing health product ang isang social product, ang resulta ay walang nasisiyahan: nawala ang social na katangian, at walang gumagamit ng sports function.

At tulad ng mga naunang pagbabago: **Walang abiso bago ang pagbabago, walang opsyon pagkatapos ng pagbabago.**

---

## Pangunahing Problema: Walang Pahintulot ng User

Ang tatlong pagbabago sa itaas, bawat isa kung ihiwalay ay maaaring isipin ng ilan na "okay lang naman, hindi naman ganoon kaseryoso." Ngunit pagsama-samahin ang mga ito:

1. Pagbawas ng visibility ng likes
2. Pagtanggal ng kaibigan = pagtanggal ng likes
3. Pagkansela ng pagbibigay ng QuanYiKa

**Tatlong pagbabago, walang isa man lang na nag-abiso nang maaga sa user. Walang isa man lang na nagbigay ng karapatang pumili sa user. Walang isa man lang na nagpapahintulot sa user na panatilihin ang dating function.**

Ang opisyal na paliwanag ng Xiaotiancai ay "ginawa ang optimization batay sa feedback ng ilang magulang."

Feedback ng ilang magulang.

Sa madaling salita, kung paano babaguhin ang produkto ay nakadepende sa "kung sino ang malakas ang boses." Isang grupo ng magulang ang nagsabing "ayaw kong makita ng anak ko ang mga like ng estranghero," sige, baguhin. Kung ano ang iniisip ng mga bata na aktuwal na gumagamit ng function na ito — hindi mahalaga.

Ang device ay ginagamit ng iyong anak, hindi ng mga "magulang" na iyon.

---

## Bakit Ito Ay "Hindi Pagpapahalaga sa User"

Ang tatlong pagbabagong ito ay may parehong katangian:

1. **Walang anunsyo** — maaaring hindi man lang nabanggit sa update notes
2. **Walang pagpipilian** — binago na, tanggapin mo man o hindi
3. **Walang kompensasyon** — ang social assets na naipon ng user (bilang ng likes, QuanYiKa) ay direktang bumaba ang halaga o nawala

Kung ito ay isang ordinaryong maliit na kumpanya, masasabi mong "hindi nila alam." Ngunit ang Xiaotiancai ay may malaking bahagi ng merkado ng relo ng bata, at sasabihin mo bang hindi nila alam ang product management at user communication? Hindi sila marunong, kundi wala silang pakialam.

Sa kanilang pananaw:
- Ang tunay na user ng relo ay ang **magulang** (ang nagbabayad)
- Ang bata ay **gumagamit lamang** (walang kapangyarihang magdesisyon)
- Kaya basta hindi magreklamo ang magulang, okay lang kahit ano ang iisipin ng bata

Ang lohikang ito ay maaaring makatwiran sa negosyo, ngunit hindi ito makatwiran sa moral.

---

## Huling Tala

Ang artikulong ito ay isinulat noong 2026. Kung matagal na ang lumipas nang mabasa mo ito, maaaring ibinalik na ng Xiaotiancai ang mga pagbabago, o maaaring lalo pang pinalala.

Kahit ano pa man, ang lohika sa likod ng mga pagbabagong ito ay hindi nagbago: **Hindi pag-aari ng user ang kanilang datos, hindi pag-aari ng user ang kanilang social relationships, hindi pag-aari ng user ang kanilang mga nakuhang karapatan. Pag-aari ng Xiaotiancai ang lahat ng ito.**
