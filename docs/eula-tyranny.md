# 用户协议霸王条款分析

> 以下内容基于小天才用户协议公开版本整理。如果你手上有最新版本发现有变化，欢迎 PR 更新。

---

## 先说结论

小天才的用户协议本身不是什么行业里最恶劣的。但它有几个非常典型的问题，值得拿出来说。

这些问题放在一起看，意思很明白：

**"你买了我们的设备，但这设备怎么用、你能不能继续用，我们说了算。"**

---

## 1. 单方面修改协议的权利

很多用户协议都有这条，小天才也不例外。大概意思就是：

> 公司有权随时修改本协议，修改后的协议一经发布即生效。

这在法律上叫"单方变更权"。虽然实践中法院不一定支持这种条款（民法典合同编有规定，格式条款提供方不能不合理地免除自己责任、加重对方责任），但对普通用户来说，谁会因为协议更新了去起诉？

实际上就是：**他们改了什么你可能永远不知道，但你只要继续用就是"同意"了。**

相比之下，一些做得好的服务会在重大变更时主动通知用户、甚至要求重新确认。小天才没这么做。

## 2. 数据收集的范围

儿童智能设备的数据收集问题比普通手机敏感得多。因为用户是未成年人。

小天才手表收集的数据包括但不限于：
- 实时定位（GPS/WiFi/基站）
- 通讯录和通话记录
- 微聊消息内容
- 设备的全部使用行为

这些对儿童来说属于敏感个人信息。小天才的隐私政策里虽然会列出来收集了什么，但问题是：

- 收集的必要性是否充分？
- 数据存储和传输的加密措施是否到位？
- 数据会不会被用于其他目的？

这些问题外部审计不了，因为没有公开透明的安全报告。

## 3. 服务终止权

协议里有条款写明，公司可以在某些情况下终止服务。原话大意是：

> 如用户违反本协议，公司有权随时终止服务，并不退还已支付费用。

这条的问题在于"违反本协议"的界定权完全在公司手里。说你违规你就违规，不需要第三方裁决。

如果你给孩子买了手表，充了年费，然后某天公司说你违规了——服务停了，钱也不退。你有什么办法？

## 4. 协议里没说的东西（这可能才是最大的问题）

用户协议里没有明确写的是：

- 设备具有远程管控能力，可以远程锁定或限制功能
- 安装未经许可的第三方应用会被封号
- 尝试 root 设备同样会导致封号

这些内容在协议里找不到明确条款。但实际执行中是存在的。

**一个不在协议里写明的、会导致服务终止的规则，用户根本无从知晓。** 这在合同法上叫"未告知的重要条款"，能不能作为有效约定是很成问题的。

---

## 补充说明

不是说要小天才搞什么法外开恩。条款偏向公司本身在行业里很普遍。

但问题是：

1. 这是**儿童设备**，购买决策者和使用者是分离的（家长买、孩子用）
2. 管控力度远超普通智能设备
3. 出问题的影响不只是换个手机——孩子的通讯工具直接被切断

这些东西综合起来看，就不是"行业都这样"能解释过去的了。

### 如果你想较真

小天才用户协议里写了"如有争议，双方协商解决"。具体管辖法院或仲裁机构要看协议版本。如果你真想仔细看，可以去他们官网找当前版本的用户协议和隐私政策。

发现了什么问题，或者协议更新了内容有变化，欢迎提 Issue。
