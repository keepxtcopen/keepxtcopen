# 我该怎么做

> 光骂没用，得知道能做什么。下面按不同身份整理了能做的事。

---

## 作为普通消费者

### 1. 买之前想清楚

如果你正在考虑给孩子买小天才手表，问自己几个问题：

- **你买手表是为了什么？** 如果只是为了定位和通话，那市面上能做到这个功能的手表很多，不一定非选小天才
- **孩子班上是不是都在用小天才？** 如果是，你要做好准备——买了其他品牌，孩子可能没法通过手表加同学好友
- **你能接受设备不受自己控制吗？** 小天才的设备有远程管控能力，理论上厂商可以随时决定你的设备能干什么、不能干什么
- **你能接受哪天功能被砍了、账号被封了都没地方说理吗？** 用户协议偏向公司，维权渠道有限

这些问题没有标准答案。每个人的情况不一样。但至少应该在买之前知道这些，而不是买了之后才发现。

### 2. 如果已经买了

- **不要把家长App的权限完全交给孩子。** 小天才的设备管控最终是通过家长端实现的，家长账号是最高权限
- **定期检查用户协议和隐私政策有没有更新。** 虽然没人会天天看，但至少大版本更新时扫一眼
- **不要尝试 root 或安装非官方应用。** 在当前的政策下，这会导致封号且几乎没有申诉渠道
- **有问题先保留证据。** 截图、录屏、客服对话记录，万一走到维权那一步用得上

### 3. 考虑替代方案

如果你确定不想买小天才，市面上还有这些选择（不构成推荐，只是列举）：

- **华为儿童手表** — 生态相对开放，但社交也是仅限于华为设备
- **小米米兔手表** — 性价比路线，功能相对基础
- **360 儿童手表** — 比较老牌的玩家，市场份额不如小天才
- **步步高旗下其他品牌** — 嗯，跟小天才其实是一家的

> 说实话，目前国内儿童手表市场在社交互通上基本是各玩各的。这不是小天才一家的问题，是整个行业的问题。只不过小天才份额最大、做得最绝。

## 作为开发者 / 开源爱好者

### 1. 向 GNU FSF 举报 GPL 违规

小天才手表使用了 Linux 内核但没有公开源码。你可以向 **Free Software Foundation** 或 **Software Freedom Conservancy**（SFC）举报。

GNU 官网有专门的 GPL 合规页面：

- **GNU GPL Compliance** — https://www.gnu.org/licenses/gpl-compliance.html
- **Software Freedom Conservancy** — https://sfconservancy.org/

举报时需要提供：

- 设备型号和购买的 Android 版本
- 证明设备使用了 Linux 内核的证据（比如内核版本号截图）
- 证明厂商没有提供源码的证据（比如在官网和 GitHub 上搜索无结果）

SFC 过去成功推动过很多公司（如 LG、三星、思科）遵守 GPL。虽然他们主要关注海外市场，但中英文举报他们都受理。

如果你也想加入我们，欢迎fork本项目、提Pull Request

### 2. 在 GitHub 上关注和推进

- 搜索 "xiaotiancai kernel" 或 "imoo kernel source" 相关 issue
- 如果有开发者已经在做反编译或内核分析的工作，关注并参与
- 如果小天才某天发布了内核源码，检查其合规性

### 3. 帮助普通用户理解技术问题

- 在论坛里帮人解释什么是 GPL、为什么闭源有问题
- 翻译和传播相关技术内容
- 如果有能力，可以尝试分析设备固件并公开发现

## 作为被云控封号的用户

### 1. 先确认自己到底违反了哪条规则

说实话，很多时候用户自己都不知道为什么被封。常见的情况：

- 通过 adb 安装过 APK
- 尝试过获取 root 权限
- 绑定过bot
- 存在逆向并且调用API的行为 <-（作者经历）
- 被系统误判（较少见）

**但是关键问题是：这些"规则"在用户协议里并没有明确写出来。**

所以你首先是"被一个你不知道的规则处罚了"。

### 2. 维权渠道

**第一步：找客服**

虽然大概率没结果，但这是走其他渠道的前提——你得先证明你找过厂商协商了。

- 小天才官方客服热线：400-610-3999
- 保留所有通话录音和聊天记录

**第二步：12315 投诉**

全国统一的消费者投诉热线。可以在微信小程序搜"全国12315平台"直接投诉。

投诉要点：

- 购买的设备因厂商单方面判定违规而被限制正常使用
- 用户协议中未明确列出封号的具体规则
- 封号导致设备无法正常使用，影响了孩子的正常通讯和安全

**第三步：工信部投诉**

如果涉及通信功能被限制，或者上一步维权失败，可以向工信部电信用户申诉受理中心投诉。

网址：https://www.chinatcc.gov.cn/

**第四步：媒体曝光**

- 在黑猫投诉（https://tousu.sina.com.cn/）上发帖
- 在知乎发帖讲自己的经历
- 联系数码类博主或自媒体

小天才对品牌形象还是很在意的。单个用户维权可能没人在乎，但如果话题发酵了，他们会有压力。

### 3. 集体维权的可能性

儿童手表封号这个问题不是个例。如果你被封了，大概率还有很多人跟你一样。

可以尝试：

- 在酷安、贴吧、知乎上找同样遭遇的人
- 集体向 12315 或工信部投诉（集体投诉比单个投诉更有效）
- 如果有律师资源，可以咨询是否构成集体诉讼的条件
