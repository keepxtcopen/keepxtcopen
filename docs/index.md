## 快速了解

如果你是第一次听说这些事情，这里给你一个极简版：

**小天才做了什么：**

1. 手表系统基于安卓（Linux内核），但从未按照GPL协议要求公开内核源码 — 这在法律上是有问题的
2. 用户协议里有大量偏向公司的条款，包括单方面修改协议的权利、限制用户维权途径等
3. 设备有云端管控系统，检测到root、安装第三方App等情况会直接封禁账号，且协议中并未明确告知
4. 微聊系统只允许小天才设备之间加好友，利用市场优势地位制造社交壁垒，本质是绑架儿童社交关系
