# เหตการณ Xiaotiancai ถอดบทเรยนและแบนบญช

> บทความนประกอบดวยเนอหาท**ถายทอดหลายตอ** อาจไมถกตองทงหมด หากพบขอมลผดพลาดสามารถแจง Issue ได

เมอวนท 26 มกราคม 2026 Xiaotiancai ไดถอดวธการเจาะระบบตางๆบนโลกออนไลนออก

- ถอดวดทเกยวของกบการเจาะระบบจำนวนมากบน Bilibili, Douyin และแพลตฟอรมอนๆ
- แบนบญช Bilibili สามบญชเปนเวลา**30 ว น**
-  แบนบญช Douyin อยาง**ถาวร**อยางนอยหนงบญช

และออก**หนงสอสอสาร** (เนอหาเปนฉบบโคลน ไมใชฉบบทางการ เนองจากฉบบแทจรงไมสามารถพสจนได): ![รป](../../assets/371cecf6f4dede531f8497c33b2eaba7.png)

> ถงผทำการเจาะระบบ Xiaotiancai:
> พวกเราไมไดผด
> การโพสตบทเรยนการ Root นเปนงานอดเรกของเราเอง แต Xiaotiancai กลาวหาวามนจะทำใหผเยาว "ตด" และเปนอนตรายตอรางกายและจตใจ
> ความสำเรจของการ Root มเพยงเราเทานนทเขาใจ

ดงทเหนในภาพ Xiaotiancai ใชเหตผล "ปองกนการตดเกม" และการ "ละเมดลขสทธ" ทนาขน (ตว Xiaotiancai เองกไมไดทำตามขอกำหนดโอเพนซอรส และบางสวนเปนเพยงการสอนตดตงซอฟตแวรบคคลทสาม) ในการสงหนงสอแจง

กลาวถงสามประเดน

- ชกนำใหดาวนโหลดเกมทผานการตรวจสอบ ละเมดนโยบายปองกนการตดเกม
- เกยวของกบการละเมดลขสทธ (แตไมไดระบ**ผเสยหาย**)
- ละเมดการดำเนนการ "ความสะอาด"

กอนอน **Root** (หรอ "การเจาะระบบ" ทพดถง) ไมใชเกม ผผลตจงจบอด คณจะตดการ Root ไดหรอ? เหนชดวาไมได (หรอไมใชลกษณะทวไป)
ตอมา การทเราแกไขนาฬกาของตนเอง มนผดตรงไหน? การโพสตบทเรยนการ Root เปนวดการสอน และไมใชทกคนจะตดตงเกมในวดการสอน

#### ยนหยนในการทวงสทธ อยาเชอ "เรองไรสาระ" จงจะถกตอง
