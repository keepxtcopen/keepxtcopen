# การละเมิด GPL: เคอรเนล Android แบบปด

> ไมใชแค "พวกเขาอาจละเมด" แต "พวกเขาไมไดทำตาม GPL จรงๆ"

---

## กอนอน GPL คออะไร

GPL (GNU General Public License) เปญญญาอนญาตใหใชงานซอฟตแวรโอเพนซอรส พดงายๆคอ:

- คณสามารถใชโคดทอยภายใต GPL ไดฟร
- **แตถาคณแกไขโคดและจำหนายผลตภณฑ คณตองเปดเผยซอรสโคดทแกไขแลวดวย**
- ลเคอรเนลของ Linux ใช GPLv2 ซงเปนขอกำหนดทตองทำตาม ไมใชเรองทตอรองได

นาฬกาของ Xiaotiancai ใชระบบ Android และชนลางของ Android กคอเคอรเนล Linux หมายความวา ถาคณใชเคอรเนล Linux ในการทำผลตภณฑและขายออกไป คณตองเปดเผยซอรสโคดเคอรเนล (รวมถงสวนทคณแกไข) ใหผอนไดรบ

นไมใชพนทสเทา Google เองกสงซอรสโคดเคอรเนลของ Pixel ไปไวท [kernel.google.com](https://kernel.google.com) อยางจรงจง Samsung กสง, Xiaomi กสง (แมจะชา), OPPO กสง

## Xiaotiancai ไมไดสง

ณ วนทโปรเจกตนยงคงอย Xiaotiancai ไมเคยเปดเผยซอรสโคดเคอรเนลของอปกรณใดๆเลยในชองทางสาธารณะ

ลองไปทเวบไซตของพวกเขา — หาไมเจอ
ลองคน "xiaotiancai" หรอ "imoo kernel" ใน GitHub — กหาไมเจอ
ลองคนในรายชออเมลของเคอรเนล — กไมม

นไมใชปญหา "หาไมเจอ" — เปนเพราะพวกเขาไมไดสงเลย

## ทำไมถงวาเปนการละเมด

ขอกำหนดของ GPLv2 เขยนไวอยางชดเจน:

> If you distribute copies of such a program, you must also make the source code available.

คณขายนาฬกาทตดตงเคอรเนล Linux ไป นนคอ "distribute copies" ดงนนคณตองจดหาซอรสโคด ไมมขอยกเวน

บางบรษทอาจแกตวจามวา "ใชเคอรเนลทไมไดแกไข" — กตองจดหาซอรสโคดอยด ยงไปกวานน นาฬกาของ Xiaotiancai ตองมการแกไขแนนอน — พวกเขาตองปรบแตงใหเขากบ SoC ของ Unisoc หรอโซลชนอนๆ ตองทำไดรเวอร ตองปรบแตงฮารดแวรตางๆ เปนไปไมไดทจะไมแกไข

## ทำไมถงมบรษทมากมายทไมทำตาม

นเปนปญหาในโลกความจรง ในบรรดาผ�ลตททำอปกรณ Android ในจน บรษททไมทำตาม GPL มไมใชแค Xiaotiancai รายเดยว ผ�ลตรายเลกๆและอปกรณแบบ OEM หลายรายกไมเปดเผย แตนไมไดหมายความวาการกระทำนถกตอง

เพยงแตกอนหนานไมมใครจจมองไปทประเภทนาฬกาขอเดกเทานนเอง

## นมผลกระทบอะไรจรงๆบาง

การไมเปดเผยซอรสโคดเคอรเนลหมายความวา:

1. **ไมสามารถศกษาเกยวกบความปลอดภยได** มแบลคดอร ป? มชองโหวไหม? ไมมใครตรวจสอบได
2. **ไมสามารถทำ ROM บคคลทสามได** ถาระบบเดมแย กตองทนกนไป
3. **วงจรชวตของอปกรณถกควบคมโดยผผลตแตเพยงผเดยว** ถาผผลตบอกวาหยดอปเดต อปกรณของคณกตาย
4. **อยากแกบกดวยตนเอง?** ไมมทาง

อปกรณสำหรบเดก แตเปนกลองดำ ไมอนญาตใหใครตรวจสอบโคดชนลางของมน
