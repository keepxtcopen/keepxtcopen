# การควบคมผานคลาวดและการแบนบญช

> นอาจเปนจดท**ยอมรบไดนอยทสด**สำหรบอปกรณในตระกล Xiaotiancai

---

## "การควบคมผานคลาวด" คออะไร

การควบคมผานคลาวดพดงายๆกคอ — อปกรณมบรการพนหลงทสอสารกบเซฟเวอรอยางตอเนอง ผผลตสามารถสงคำสงจากคลาวดใหอปกรณดำเนนการตางๆได

ในกรณของนาฬกาขอมอเดก กลไกนสามารถทำไดรวมถง:

- ปองกนฟงกชนจากระยะไกล
- ลบแอปพลเคชนจากระยะไกล
- ลอคอปกรณจากระยะไกล
- แบนบญช (อปกรณกลายเปนกอนอฐ)

ความสามารถเหลานตวของมนเองไมไดเลวรวยทงหมด — ตัวอยางเชน พอแมควบคมการใชนาฬกาผานแอปปกต กใชกลไกคลายๆกน

ปญหาคอ: **เมอผผลตใชกลไกนในการลงโทษผใช**

---

## รปแบบปจจบนของการควบคมผานคลาวด

อปกรณทถกควบคมผานคลาวดจะไมกลายเปนกอนอฐทนท จรงๆแลวการลงโทษของ Xiaotiancai แบงเปน**ระดบชน** โดย "การละเมด" ประเภทและความรนแรงตางกนจะถกลดรอบตางกนไป ดงนคอขอกำกดทพบไดบอยในปจจบน:

### การพกการพดใน Micro Chat

ผใชจะเหนขอกำกดการพดเมอเปด Micro Chat:

```plaintext

เนอหาทโพสตเมอไมนานนละเมดขอบฏบตชมชน Xiaotiancai
```

จากนนฟงกชน Micro Chat ถกจำกด ไมสามารถสงขอความ ไมสามารถตอบได แตทแปลกคอ — "ขอบฏบตชมชน" ทวานมเนอหาอะไรบาง ขอความไหนทละเมด ระยะเวลาในการลงโทษนานแคไหน — ไมมการบอกคณเลย

คณไมไดตดตอกบกฎทโปรงใส แตกำลงตดตอกบกลองดำ

### ไมสามารถโพสตใน Friend Circle

ฟงกชน Friend Circle ถกจำกดบางสวน ลกษณะเฉพาะคอ:

- ไมเหนอปเดตของ Friend Circle
- ตนเองไมสามารถโพสตเนอหาใหมได
- แตเนอหาทโพสตไปแลวอาจยงมองเหนได

เทากบตดฟงกชนทางสงคมของทงหมด สำหรบเดกทใชนาฬกาเพอตดตอสมพนธกบเพอนในโรงเรยน ขอกำกดนทำลายประโยชนหลกในการใชงานทางสงคมของนาฬกาโดยสบท

### ฟงกชนสวนใหญขนวา "เครอขายผดปกต"

นเปนลกษณะทดมงงายทสด ไมใชวาคณตอเนเตอรอนเทอรเนตไมได แตฟงกชนเฉพาะขนวาเครอขายผดปกต:

- บางหนาโหลดไมปรากฏ
- เมอกดเขาฟงกชนขนวา "การเชอมตอเครอขายผดปกต กรุณาลองใหมภายหลง"
- แตฟงกชนอนๆบนอปกรณเดยวกนอาจทำงานไดตามปกต

ขอกำกดแบบ "นม" นแยกวาการแบนโดยตรง — คณไมแนใจวาเปนปญหาทอปกรณ ปญหาเครอขาย หรอถกจำกด

### เซฟเวอรสนรหสขอผดพลาดเฉพาะ

สำหรบผใชทมพนฐานทางเทคนค สามารถจบแพคเกจเพอดสญญาณการแบนทชดเจนชนได อปกรณทถกจำกดจะไดรบการตอบสนองตอไปนเมอเรยกใช API:

```json
{"code":"000007","desc":"request parameter is invalid."}
```

"พารามเตอรคำขอไมถกตอง" — ดเหมอนวาสงพารามเตอรผด แตความจรงแลวนคอการตอบสนองการปฏเสธแบบรวมของเซฟเวอรตอบญชทถกจำกด

ไมใชวาพารามเตอรไมถกตองจรงๆ แตเซฟเวอรตดสนใจวาคำขอนไมสมควรไดรบการดำเนนการ จงสงขอมลผดพลาดทคลมเครอกลบไป

การออกแบบนเปนไปโดยตงใจ: **ไมใหขอมลผดพลาดทชดเจน เพอใหผใชคดวาเปนปญหาทอปกรณของตนเอง ไมใชถกจำกด**

## กรณจรงของการแบนบญช

มผใช Xiaotiancai จำนวนมากในโลกออนไลนทราบถงเหตการณตอไปน:

**กรณ A: ตดตงแอปบคคลทสามแลวถกลอค**
ผใชตดตงแอปทไมใชจากรานทางการใหกบนาฬกาผานบางวธการ (adb หรอวธอนๆ) แลว หรอเมอเวลาผานไป นาฬกากขนขอกำกดวาบญชละเมดขอตกลง และฟงกชนของอปกรณถกจำกด ไมสามารถใชโทรออก พดคย ฯลฯ ได

**กรณ B: ถกแบนหลงจาก root**
ผใชพยายามเขาถงสทธ root ของอปกรณ เพอลบซอฟตแวรทตดตงไวจากโรงงาน หรอปรบปรงระบบ เซฟเวอรตรวจพบและแบนบญช

**กรณ C: ไมสามารถเปดใชงานหลงจากการตงคาโรงงาน**
ผใชบางคนพบวาเมอตงคาโรงงานและพยายามเปดใชงานครงใหม ไดรบขอกำกดวา "อปกรณนถกผกไวแลว" ขนตอนการแกปญหากบฝายบรการลกคามความยงยากมาก

---

## ประเดนสำคญของปญหา

เกยวกบการแบนบญช มประเดนหลกสามประการ:

### 1. ไมไดระบไวในขอตกลง

ตามทกลาวไวกอนหนา ในขอกำหนดการใชงาน**ไมมขอกำหนดทชดเจน**วา "การ root จะนำไปสการแบนบญช" หรอ "การตดตงแอปพลเคชนบคคลทสามจะนำไปสการแบนบญช" ผใชละเมดกฎโดยไมรวากฎมอ ย

ถากฎหนงจะทำใหอปกรณทคณซอดวยเงนหลายรอยถงหลายพนหยวนกลายเปนกอนอฐ อยางนอยกฎนกควรเขยนไวชดเจนเปนตวจรง

### 2. ความเปนเจาของอปกรณ vs สทธในการใช

คณซอนาฬกาเรอหนง ในทางกายภาพมนเปนของคณ แตกลไกการควบคมผานคลาวดของ Xiaotiancai หมายความวา: **พวกเขาสามารถตดสินจากระยะไกลไดวาอปกรณของคณสามารถใชงานไดตามปกตหรอไม**

ซงมนกเลกซบซอน — สงทคณจายเงนไป จรงๆแลวคอ "อปกรณทเปนของคณ" หรอ "อปกรณทไดรบอนญาตใหใช"?

### 3. ไมมชองทางในการอบรม

หลงจากการแบนบญช ผใชสวนใหญไมมชองทางในการอบรมทมประสทธภาพ การตดตอฝายบรการลกคา = ตอคอย สงตอ ปดความรบผชอบ ลองคนกระทในเกยวกบการแบนบญชของ Xiaotiancai ในอนเทอรเนต — สวนใหญไมไดรบการแกไขทนำพอใจ

## ทำไมพวกเขาถงทำแบบน

พดอยางยตธรรม Xiaotiancai ม "เหตผล" ของพวกเขา:

- รกษาความปลอดภยของอปกรณ (ปองกนซอฟตแวรไมประสงค)
- ปองกนการเปดฟงกชนทพอแมไมตองการ
- รกษาความมนคงคงทของระบบ (ถาอปกรณเดกเกดปญหา พอแมจะรองเรยน)

แต "เหตผล" เหลานจะถกตองหรอไม กเปนคนละเรองกบ "การกระทำนสมเหตสมผลหรอไม"

ปญหาคอ:

ระบบ Android มกลไกความปลอดภย (แซนดบ็อกซของแอป, ระบบสทธ) ในการแยกความปลอดภยอยแลว การใชการควบคมผานคลาวดเปนวธการควบคมหลก จรงๆแลวคอ**การควบคมในมตเพมเตม** — อยเหนอกวาการจดการสทธของระบบปฏบตการ ตงระบบขนมาใหมบนคลาวด

คณไมจำเปนตองมอมอของผใชไวดงๆเพอปองกนเขาไมใหทำผด

## ทำไมเรองนถงเลวรายเปนพเศษ

ผใชนาฬกาขอมอเดกคอเดก แตผซอคอพอแม ความตองการหลกของพอแมเมอซออปกรณนคออะไร?

1. ตดตอกบลกได
2. รวาลกอยทไหน
3. ใหลกใชอยางมนใจ

ถาเนองจากพฤตกรม "ละเมด" ทพอแมอาจไมรเรองดวยเลย นาฬกากลายเปนกอนอฐ — **ลกตดตอพอแมไมได พอแมกรวาลกอยทไหนไมได**

นคอการเอาความปลอดภยของลกเปนตวตอรอง
