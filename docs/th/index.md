## ทำความรวดๆ

ถาคณเพงเคยไดยนเรองพวกนเปนครงแรก นคอเวอรสนแบบสนๆ:

**Xiaotiancai ทำอะไรลงไป:**

1. ระบบนาฬกาขอระบบปฏบตการ Android (เคอรเนล Linux) แตไมเคยเปดเผยซอรสโคดเคอรเนลตามขอกำหนดของ GPL — ซงถอวาผดกฎหมาย
2. ขอกำหนดการใชงานมขอความจำนวนมากทเออประโยชนตบบรษท รวมถงสทธในการแกไขขอกำหนดเพยงฝายเดยว การจำกดชองทางทวงสทธของผใช ฯลฯ
3. อปกรณมระบบควบคมผานคลาวด เมอตรวจพบการ root หรอการตดตงแอปบคคลทสาม จะแบนบญชทนท โดยไมไดแจงไวชดเจนในขอกำหนด
4. ระบบ Micro Chat อนญาตใหเพมเพอนไดเฉพาะอปกรณ Xiaotiancai เทานน ใชสถานะผนำตลาดสรางกำแพงทางสงคม ซงจรงๆแลวคอการยดยงความสมพนธทางสงคมของเดก
