# ฉนควรทำอยางไร

> แคดาหรออยางเดยวไมพอ ตองรวาทำอะไรไดบาง ดานลางนรวบรวมสงททำไดตามสถานะของแตละคน

## ในฐานะผบรโภคทวไป

### 1. คดใหรอบคอบกอนซอ

ถาคณกำลงพจารณาซอนาฬกาขอมอ Xiaotiancai ใหลก ถามตวกนเอง:

- **คณซอนาฬกามาเพออะไร?** ถาเพอการหาตำแหนงและโทรออกอยางเดยว มนาฬกาหลายรนในตลาดททำได เหมอนกน ไมจำเปนตองเลอก Xiaotiancai
- **ในชนเรยนของลกใช Xiaotiancai กนหมดหรอเปล?** ถาใช คณตองเตรยมตว—ถาซอยหอยางอน ลกอาจไมสามารถเพมเพอนผานนาฬกาได
- **คณยอมรบไดหรอไมทอปกรณไมไดอยภายใตการควบคมของคณ?** อปกรณของ Xiaotiancai มความสามารถในการควบคมทางไกล ตามทฤษฎแลวผผลตสามารถตดสินไดตลอดเวลาวาอปกรณคณทำอะไรไดบางและทำอะไรไมไดบาง
- **คณยอมรบไดหรอไมวาวนหนงฟงกชนอาจถกตดหรอบญชอาจถกแบน แลวไมมทจะรองเรยน?** ขอกำหนดการใชงานเออประโยชนตบบรษท ชองทางการทวงสทธจำกด

คำถามเหลานไมมคำตอบตายตว สถานการณแตละคนไมเหมอนกน แตอยางนอยคณควรรสร.องเหลานกอนซอ ไมใชเพงมาพบหลงจากซอแลว

### 2. ถาซอไปแลว

- **อยาใหสทธผปกครองของแอปแกเดก** การควบคมอปกรณ Xiaotiancai สวนใหญทำผานฝายผปกครอง บญชผปกครองคอสทธสงสด
- **ตรวจสอบขอกำหนดการใชงานและนโยบายความเปนสวนตวเปนระยะๆวามการอปเดตหรอไม** ถงแมไมมใครอานทกวน แตอยางนอยเมออปเดตเวอรชนใหญกควรพอด
- **อยาพยายาม root หรอตดตงแอปทไมใชทางการ** ภายใตนโยบายปจจบน นจะนำไปสการแบนบญช และแทบไมมชองทางอบรม
- **เกบหลกฐานไวกอนหากเกดปญหา** ภาพหนาจอ คลปวิดโอ บนทกการสนทนากบฝายบรการลกคา เผอวาในภายหลงตองใชตอนทวงสทธ

### 3. พจารณาทางเลอกอนๆ

ถาคณตดสนใจแลววาไมตองการซอ Xiaotiancai ในตลาดยงมทางเลอกตอไปน (ไมใชการแนะนำ เปนเพยงการระบชอ):

- **Huawei Children's Watch** — นเวศคอนขางเปดกวา แตทางสงคมกจำกดเฉพาะอปกรณ Huawei เชนกน
- **Xiaomi Mi Rabbit Watch** — แนวทางคมคา ฟงกชนพนฐาน
- **360 Children's Watch** — ผเลนทมานาน สวนแบงตลาดนอยกวา Xiaotiancai
- **แบรนดอ นๆในเครอง BBK** — เออ จรงๆกเปนบรษทเดยวกบ Xiaotiancai

> พดตามตรง ในปจจบนตลาดนาฬกาขอมอเดกจน ในเรองการเชอมตอทางสงคมตางคนตางทำ นไมใชปญหาของ Xiaotiancai เพยงรายเดยว เปนปญหาของทงอตสาหกรรม เพยงแต Xiaotiancai มสวนแบงมากทสดและทำรนแรงทสด

## ในฐานะนกพฒนา / ผชอโอเพนซอรส

### 1. แจงการละเมด GPL ตอ GNU FSF

นาฬกาขอมอ Xiaotiancai ใชเคอรเนล Linux แตไมเปดเผยซอรสโคด คณสามารถแจงไปยง **Free Software Foundation** หรอ **Software Freedom Conservancy (SFC)**

หนาเวบ GPL Compliance ของ GNU มรายละเอยดโดยเฉพาะ:

- **GNU GPL Compliance** — https://www.gnu.org/licenses/gpl-compliance.html
- **Software Freedom Conservancy** — https://sfconservancy.org/

ในการแจงตองใหขอมล:

- รนของอปกรณและรน Android ทซอมา
- หลกฐานวาอปกรณใชเคอรเนล Linux (เชน ภาพหนาจอเลขรนเคอรเนล)
- หลกฐานวาผผลตไมไดจดหาซอรสโคด (เชน ไมพบแหลงในเวบไซตและ GitHub)

กอนหนาน SFC ประสบความสำเรจในการผลกดนบรษทหลายแหง (เชน LG, Samsung, Cisco) ใหทำตาม GPL แมพวกเขาจะมงเนนตลาดตางประเทศเปนหลก แตรบขอมลการแจงทงภาษาจนกลางและภาษาองกฤษ

ถาคณตองการรวมกบเรา ยนดตอนรบให Fork โปรเจกตนและสง Pull Request

### 2. ตดตามและผลกดนบน GitHub

- คนหา issue ทเกยวของกบ "xiaotiancai kernel" หรอ "imoo kernel source"
- หากมนกพฒนาทำงานเกยวกบการดีคอมไพลหรอวเคราะหเคอรเนลอยแลว ตดตามและรวมมอกน
- หากวนหนง Xiaotiancai ปลอยซอรสโคดเคอรเนล ตรวจสอบความถกตอง

### 3. ชวยผใชทวไปเขาใจปญหาทางเทคนค

- ชวยอธบายในเวบบอรดาวา GPL คออะไร ทำไมการปดซอรสโคดถงเปนปญหา
- แปลและเผยแพรเนอหาทางเทคนคทเกยวของ
- หากมความสามารถ ลองวเคราะหเฟรมแวรของอปกรณและเปดเผยขอมลทคนพบ

## ในฐานะผใชทถกแบนบญชผานคลาวด

### 1. ขนแรก หาวาคณละเมดกฎขอไหน

จรงๆแลวผใชสวนใหญไมรตวาตวเองถกแบนเพราะอะไร กรณทพบบอย:

- ตดตง APK ผาน adb
- พยายามเขาถงสทธ root
- เคยผกบอท
- มการใชวศวกรรมยอนกลบและเรยกใช API (ประสบการณของผเขยน)
- ถกเซฟเวอรวนจฉยผด (พบไดนอย)

**แตปญหาสำคญคอ: "กฎ" เหลานไมไดเขยนไวอยางชดเจนในขอกำหนดการใชงาน**

ดงนนคณจงถกลงโทษดวย "กฎทคณไมร"

### 2. ชองทางการทวงสทธ

**ขนตอนแรก: ตดตอฝายบรการลกคา**

แมสวนใหญจะไมไดผล แตนคอขนตอนกอนใชชองทางอน คณตองพสจนใหไดวาคณไดตดตอหาผผลตเพอเจรจาแลวกอน

- สายดวนฝายบรการลกคา Xiaotiancai: 400-610-3999
- เกบบนทกการสนทนาทางโทรศพทและขอมสทกอยาง

**ขนตอนทสอง: องเรยนตอ 12315**

สายดวนองเรยนผบรโภคทวประเทศ คณสามารถคนหา "แพลตฟอรม 12315 ทวประเทศ" ในมนิโปรแกรม WeChat เพอองเรยนโดยตรง

ประเดนในการองเรยน:

- อปกรณทซอมาไมสามารถใชงานไดตามปกต เนองจากผผลตตดสนใจโดยฝายเดยววาละเมด
- ในขอกำหนดการใชงานไมไดระบกฎเฉพาะสำหรบการแบนบญช
- การแบนบญชทำใหอปกรณไมสามารถใชงานไดตามปกต สงผลกระทบตอการสอสารและความปลอดภยของเดก

**ขนตอนทสาม: องเรยนตอ กระทรวงอตสาหกรรมและเทคโนโลยสารสนเทศ**

หากเกยวกบการจำกดฟงกชนการสอสาร หรอการทวงสทธในขนตอนทสองไมสำเรจ สามารถองเรยนตอศนยรบองเรยนขารองผใชบรการโทรคมนาคมของกระทรวงอตสาหกรรมและเทคโนโลยสารสนเทศ

เวบไซต: https://www.chinatcc.gov.cn/

**ขนตอนทส: เปดเรองในสอ**

- โพสตใน Black Cat Complaint (https://tousu.sina.com.cn/)

Xiaotiancai ใหความสำคญกบภาพลกษณของแบรนดมาก การทวงสทธของผใชเดยวอาจไมมใครสนใจ แตเมอประเดนเรมเปนทพดถง พวกเขาจะกดดน

### 3. ความเปนไปไดในการรวมตวจณะ
