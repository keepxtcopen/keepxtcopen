# 吾當如何

> 光罵無用，須知能為何事。下按不同身份整理所能為者。

## 作為普通消費者

### 一、購前深思

若君正考慮為孩子購小天才手錶，先自問數端：

- **爾購手錶欲為何用？** 若僅為定位與通話，市面上能行此功能之手錶甚多，不必非選小天才
- **孩子班上是否皆用小天才？** 若是，君須做好準備——購其他品牌，孩子或無法通過手錶加同學好友
- **爾能接受設備不受己控乎？** 小天才設備有遠程管控能力，理論上廠商可隨時決定爾之設備能作何事、不能作何事
- **爾能接受他日功能被砍、賬號被封而無處說理乎？** 用戶協議偏袒公司，維權渠道有限

此等問題無標準答案。每人情形不同。然至少應在購前知曉，而非既購之後方始發現。

### 二、若已購之

- **勿將家長App之權限完全交予孩子。** 小天才設備管控終通過家長端實現，家長賬號為最高權限
- **定期檢查用戶協議及隱私政策有無更新。** 雖無人會日日查看，然至少大版本更新時掃一眼
- **勿嘗試root或安裝非官方應用。** 在當前政策下，此將致封號且幾無申訴渠道
- **有問題先保留證據。** 截圖、錄屏、客服對話記錄，萬一走到維權那一步用得著

### 三、考慮替代方案

若君確定不欲購小天才，市面上尚有此等選擇（非推薦，僅列舉）：

- **華為兒童手錶** — 生態相對開放，然社交亦限於華為設備
- **小米米兔手錶** — 性價比路線，功能相對基礎
- **360兒童手錶** — 較老牌之參與者，市場份額不如小天才
- **步步高旗下其他品牌** — 嗯，與小天才實為一家

> 老實說，當前國內兒童手錶市場在社交互通上基本各行其是。此非小天才一家之問題，乃整個行業之問題。特以小天才份額最大、做得最絕耳。

## 作為開發者 / 開源愛好者

### 一、向GNU FSF舉報GPL違規

小天才手錶使用Linux內核而未公開源碼。君可向**Free Software Foundation**或**Software Freedom Conservancy**（SFC）舉報。

GNU官網有專門之GPL合規頁面：

- **GNU GPL Compliance** — https://www.gnu.org/licenses/gpl-compliance.html
- **Software Freedom Conservancy** — https://sfconservancy.org/

舉報時須提供：

- 設備型號及所購之Android版本
- 證明設備使用Linux內核之證據（如內核版本號截圖）
- 證明廠商未提供源碼之證據（如在官網及GitHub上搜索無結果）

SFC過去成功推動過諸多公司（如LG、三星、思科）遵守GPL。雖彼等主要關注海外市場，然中英文舉報皆受理。

若君亦欲加入吾等，歡迎fork此項目、提Pull Request。

### 二、在GitHub上關注與推進

- 搜索「xiaotiancai kernel」或「imoo kernel source」相關issue
- 若有開發者已在做反編譯或內核分析之工作，關注並參與
- 若小天才他日發布內核源碼，檢查其合規性

### 三、幫助普通用戶理解技術問題

- 在論壇中幫人解釋何為GPL、何以閉源有問題
- 翻譯和傳播相關技術內容
- 若有能力，可嘗試分析設備固件並公開發現

## 作為被雲控封號之用戶

### 一、先確認己身究竟違反何條規則

老實說，很多時候用戶自己亦不知為何被封。常見情形：

- 通過adb安裝過APK
- 嘗試過獲取root權限
- 綁定過bot
- 存在逆向且調用API之行為 <-（作者經歷）
- 被系統誤判（較少見）

**然關鍵問題在於：此等「規則」在用戶協議中並未明確寫出。**

故爾首先是「被一爾所不知之規則處罰了」。

### 二、維權渠道

**第一步：找客服**

雖大概率無結果，然此為走其他渠道之前提——爾須先證明曾找廠商協商。

- 小天才官方客服熱線：400-610-3999
- 保留所有通話錄音及聊天記錄

**第二步：12315投訴**

全國統一之消費者投訴熱線。可在微信小程序搜「全國12315平臺」直接投訴。

投訴要點：

- 所購設備因廠商單方面判定違規而被限制正常使用
- 用戶協議中未明確列出封號之具體規則
- 封號導致設備無法正常使用，影響了孩子之正常通訊與安全

**第三步：工信部投訴**

若涉及通信功能被限制，或上一步維權失敗，可向工信部電信用戶申訴受理中心投訴。

網址：https://www.chinatcc.gov.cn/

**第四步：媒體曝光**

- 在黑貓投訴（https://tousu.sina.com.cn/）上發帖
- 在知乎發帖講述己身經歷
- 聯繫數碼類博主或自媒體

小天才對品牌形象頗為在意。單個用戶維權或無人關注，然若話題發酵，彼等會有壓力。

### 三、集體維權之可能性

兒童手錶封號此問題非個例。若爾被封，大概率尚有諸多與爾相同之人。

可嘗試：

- 在酷安、貼吧、知乎上尋同樣遭遇之人
- 集體向12315或工信部投訴（集體投訴比單個投訴更有效）
- 若有律師資源，可諮詢是否構成集體訴訟之條件
