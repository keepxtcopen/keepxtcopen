# 社交壟斷與生態壁壘

> 此非技術問題，乃社會問題也。

---

## 微聊之封閉生態

小天才手錶中有一核心功能名曰「微聊」（不同版本或有異名，本質為手錶上之社交功能）。

此功能之特點：

**小天才設備只能加小天才設備為好友。**

非爾主動選擇不加其他品牌——乃技術上即不令爾加。

君用360手錶？加不了。
君用華為手錶？加不了。
君用小米米兔？加不了。

微聊之整個好友系統及通訊協議皆封閉，惟小天才設備方能相互通信。

---

## 市場份額之現實

此處不做精確數據引用（無官方數據，第三方調研機構之數字亦不統一），然有一公開事實：

**小天才在中國兒童電話手錶市場之份額長期居絕對領先地位。**

領先幾何？在諸多學校中，十個戴手錶之孩子裡，七八個為小天才。

當一品牌在垂直市場中佔據如此份額，復加一封閉社交系統，會發生何事？

## 不買小天才 = 被排除於社交圈外

此事說來荒誕，然正在真實發生。

爾孩子之同學全用小天才之微聊建群聊天。爾孩子用其他品牌之手錶，加不了群，收不到消息，看不到同學分享之物。

一次兩次尚可，日久孩子會作何想？

「爸爸/媽媽，吾亦欲購小天才。」

此非因小天才之產品有多好。乃因**社交關係被挾持矣。**

或曰此乃市場選擇之結果。然當一品牌利用封閉生態以固化己身之市場優勢地位時，此已非市場競爭矣。

## 非小天才獨創

老實說，封閉生態此套玩法非小天才首創。

Apple之iMessage亦為封閉，綠色氣泡 vs 藍色氣泡在歐美亦有社交排斥之問題。

然區別在於：

- iPhone本身非必需品，而兒童手錶對諸多家長而言乃剛需（安全需求）
- iMessage不能跨平臺，然iPhone用戶至少能用WhatsApp、Telegram、微信。而小天才微聊之替代方案為**零**
- 兒童社交比成人更易受「從眾壓力」之影響

## 「則君購其他品牌不亦宜乎？」

此言說來輕巧。

當市場第一名之份額大到一定程度，加上封閉社交此一武器，消費者實已無真正之選擇。

爾不買小天才 -> 孩子無法與同學社交 -> 孩子不悅 -> 家長被迫買小天才

此鏈條非自由之消費選擇，乃封閉生態所帶來之**鎖定效應**。

---

## 個人之見

非謂小天才當被拆除。封閉生態本身非原罪。

然當一品牌同時在多個維度（用戶協議、內核閉源、雲控、生態壁壘）皆行不利消費者之事時，此等問題便疊加起來：

- 彼用封閉生態鎖住用戶 -> 爾不得不買
- 既買之後彼用雲控限制爾 -> 爾不能按己意使用設備
- 內核不開源 -> 無任何第三方能助爾繞過此等限制
- 協議條款偏袒公司 -> 爾維權幾無可能

**每一步皆為精心設計之閉環。**
