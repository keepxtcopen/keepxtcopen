## 速覽

若君初聞此事，請觀其略：

**小天才所為：**

1. 手錶系統基於Android（Linux內核），然從未依GPL協議公開內核源碼——此法理有虧
2. 用戶協議中含大量偏袒公司之條款，包括單方修改協議之權、限制用戶維權途徑等
3. 設備有雲端管控系統，檢測root、安裝第三方應用等情形即封禁賬號，而協議中未曾明告
4. 微聊系統僅容小天才設備間互加好友，藉市場優勢地位製造社交壁壘，實為挾持兒童社交關係
